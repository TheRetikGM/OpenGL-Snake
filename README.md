# OpenGL Snake
Simple and quick implementation of game Snake, written in C++ and OpenGL

## In-Game example
![OpenGL-Snake example](/doc/example.png "InGame example")

## Controls
> **Arrows** for movement <br>
> **ESC** for pause menu

## Build and run
    git clone https://github.com/theretikgm/opengl-snake.git
    cd opengl-snake
    ./configure.sh
    ./build.sh
    ./run.sh
